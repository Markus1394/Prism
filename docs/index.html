PRISM Docs

Creating Templates

Watermark templates in PRISM are described in JSON format. This allows flexible customization of appearance, offsets, and element positioning without writing application code.

Base Structure

Every template starts with an orientation (layout), general offsets (padding), and a list of elements (elements).

JSON 
{
  "layout": "horizontal",
  "padding": 24,
  "elements": [
  ]
}

Template Parameters

KeyTypeDescriptionValues
layoutstringContainer orientation"horizontal", "vertical"
paddingintEdge padding (dp)0 - 100
elementsarrayList of elementsArray of objects

Element Types

1. Logo (logo)

JSON 
{
  "type": "logo",
  "size": 48,
  "marginEnd": 16
}

2. Main Text (main_text)

JSON 
{
  "type": "main_text",
  "size": 18,
  "bold": true,
  "marginBottom": 4
}

3. EXIF Data (exif_text)

JSON 
{
  "type": "exif_text",
  "size": 12,
  "alpha": 0.7
}

4. Divider (divider)

JSON 
{
  "type": "divider",
  "width": 2,
  "height": 40,
  "marginEnd": 16,
  "alpha": 0.3
}

5. Group (text_group)

Used for vertical grouping of elements within a horizontal layout.

JSON 
{
  "type": "text_group",
  "children": [
    { "type": "main_text", "size": 18 },
    { "type": "exif_text", "size": 12 }
  ]
}

Template Examples

Example 1: Standard

Logo on the left, divider, then text.

JSON 
{
  "layout": "horizontal",
  "padding": 24,
  "elements": [
    {
      "type": "logo",
      "size": 48,
      "marginEnd": 16
    },
    {
      "type": "divider",
      "width": 2,
      "height": 40,
      "marginEnd": 16,
      "alpha": 0.3
    },
    {
      "type": "text_group",
      "children": [
        {
          "type": "main_text",
          "size": 18,
          "bold": true
        },
        {
          "type": "exif_text",
          "size": 12,
          "marginTop": 4,
          "alpha": 0.7
        }
      ]
    }
  ]
}

Example 2: Minimal

Simple layout without dividers. Logo and text side by side.

JSON 
{
  "layout": "horizontal",
  "padding": 20,
  "elements": [
    {
      "type": "logo",
      "size": 32,
      "marginEnd": 12
    },
    {
      "type": "text_group",
      "children": [
        {
          "type": "main_text",
          "size": 15,
          "bold": true
        }
      ]
    }
  ]
}

Example 3: Classic

Text on the left, logo on the right. Parameter weight: 1 makes the text take up all available space.

JSON 
{
  "layout": "horizontal",
  "padding": 24,
  "elements": [
    {
      "type": "text_group",
      "weight": 1,
      "children": [
        {
          "type": "main_text",
          "size": 20,
          "bold": true
        },
        {
          "type": "exif_text",
          "size": 11,
          "marginTop": 6
        }
      ]
    },
    {
      "type": "logo",
      "size": 56,
      "marginStart": 24
    }
  ]
}

Example 4: Vertical (Sidebar)

For watermarks placed on the left or right side. Use rotation: 90.

JSON 
{
  "layout": "vertical",
  "padding": 16,
  "elements": [
    {
      "type": "logo",
      "size": 40,
      "marginBottom": 12
    },
    {
      "type": "main_text",
      "size": 14,
      "bold": true,
      "rotation": 90
    },
    {
      "type": "exif_text",
      "size": 10,
      "marginTop": 12,
      "rotation": 90,
      "alpha": 0.7
    }
  ]
}

How to Add a Template

You can add a template in two ways:

Method 1: From File

  1. Create a file with the .json extension on your phone.
  2. Copy the code from an example you like into it and save.
  3. In the app, tap Style → + Add Template → Load from file.

Method 2: Manually

  1. Tap the Copy button above a code example.
  2. In the app, tap + Add Template.
  3. Paste the code into the Template JSON field.
  4. Enter a name and tap Add.
